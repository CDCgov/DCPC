Develop a Machine Learning Solution

The machine learning approach consists of four major stages.

  1. Preparing a gold standard of annotations of text for the semantic entities that need to be computed in the production system
  2. Developing a NLP processing pipeline that computes, for each word in the gold standard texts, a set of features (aka attributes)
  3. Passing the list of words and their features in a BIO file to the machine learner which then trains a Language Model. The Language Model is a statistical representation of the words in the semantic classes and their surrounding context.
  4. Testing the LM against the gold standard to learn where it makes mistakes and then repeating iteratively corrections to the modelling process until the results reach an acceptable level. That level will be determined by the use case for the production system.
