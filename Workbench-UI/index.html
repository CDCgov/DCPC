Welcome to CLEW

An open platform to provide clinical Natural Language Processing (NLP) services as well as to develop, experiment with and refine clinical NLP models

The CLEW Vision & Roadmap

The purpose of the CLEW is to provide the NLP Community with an open platform to develop new processing sources for clinical texts and deploy those services to make them publicly available.

CLEW provides

  • Open source NLP and machine learning tools
  • A platform to...
    • Share new tools with the wider Clinical NLP community
    • Assemble NLP tools into a processing workflow
    • Generate training files for feeding machine learning algorithms to develop language models
  • An environment for installing pipelines as services for use by other members of the Clinical NLP community
